## 🛡️ Trunk Armor System STANDALONE

A lightweight and immersive **FiveM armor system** that allows players to retrieve armor directly from vehicle trunks using a **3rd eye interaction** and a clean **OX Lib menu**.

This script adds realistic roleplay by requiring players to physically access a vehicle trunk before equipping armor.

---

### ✨ Features

* 🔍 **3rd Eye Interaction** – Target any vehicle trunk to access the armor menu.
* 📦 **Trunk Interaction** – Opens the trunk with an animation before accessing armor.
* 📋 **OX Lib Menu** – Simple and clean menu interface.
* 🛡️ **Three Armor Types**

  * **Light Armor** – Small amount of armor, fastest to equip.
  * **Medium Armor** – Half armor, medium equip time.
  * **Heavy Armor** – Full armor, longest equip time.
* ⏱️ **Progress Bars & Animations** – Realistic equip process.
* ⚙️ **Fully Configurable**

  * Armor values
  * Progress bar times
  * Animation settings
  * Enable/disable armor types
* 📦 **Standalone Script** – No framework required.
* 🔧 **Optimized & Lightweight**

---

### 📦 Dependencies

* **ox_lib**
* **ox_target**

---

### 🎮 How It Works

1. Walk up to **any vehicle trunk**.
2. Use your **3rd eye interaction**.
3. Select **Grab Armor**.
4. An **OX menu** opens with armor options.
5. Choose **Light, Medium, or Heavy Armor**.
6. A **progress bar + animation** plays while equipping.

---

### ⚙️ Configuration

The script includes a **simple config file** where you can customize:

* Armor amounts
* Equip times
* Menu options
* Animations
